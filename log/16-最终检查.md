## 六、IDEA 快速启动（摘要）

1. `powershell -ExecutionPolicy Bypass -File execution/scripts/restart-docker-mysql.ps1`
2. IDEA 打开 `hospital-backend` → Run **`HospitalBackend (prod)`**
3. 三端 `npm install && npm run dev`
4. 验证：admin/123456 @5173，patient/123456 @5175

**注意：**

- MySQL 宿主机端口 **3307**；勿 PowerShell 管道导入中文 SQL
- prod 必须带 `HOSPITAL_DB_*` 环境变量（已写入 IDEA 配置）
- 存量库补字段：`alter_operation_log.sql`

完整步骤：根目录 `README.md` 文末、`hospital-backend/IDEA-启动说明.md`
