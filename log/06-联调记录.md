| 端 | 配置 | 默认 |
|----|------|------|
| hospital-admin | `.env.development` → `VITE_USE_MOCK=false` | 直连后端 |
| hospital-user | 同上 | 直连后端 |

恢复 Mock：将 `VITE_USE_MOCK=true` 后重启 dev server。

## 启动顺序

```bash
# 1. 后端
cd hospital-backend && mvn spring-boot:run

# 2. 前端（各开一终端）
cd hospital-admin && npm run dev    # :5173
cd hospital-user && npm run dev     # :5175
cd hospital-mini && npm run dev:h5  # :5174
```

Vite proxy：`/api` → `http://localhost:8080`

## 本轮后端契约修复

- `GET/PUT /api/admin/menu/*` — 新增菜单树（原 404）
- `GET/PUT /api/admin/config/*` — 返回 `configName`/`remark`；PUT 接受表单 Map
- `GET /api/income-expense/summary`、`/{id}/trace` — 收支页加载
- `GET /api/reimbursement/{id}/detail` + approve/reject
- `GET /api/settlement/{id}/detail` + POST settle
- `IncomeExpense` 实体增加 `type`/`recordDate` JSON 别名
