| 问题 | 修复 |
|------|------|
| SQL `CLOB` 语法 MySQL 不支持 | `execution/database/hospital_his_full.sql` → `LONGTEXT` |

## 六、网关联调（8088）

```text
GET  /admin/              → 200，index.html 含 src="/admin/..."
GET  /user/               → 200
GET  /api/common/meta     → 200
POST /api/auth/login      → 200，admin 登录成功
GET  /                      → 302 → /user/
```

启动命令：

```powershell
# 后端（示例）
$env:HOSPITAL_DB_URL='jdbc:mysql://127.0.0.1:3307/hospital_his?...'
$env:HOSPITAL_DB_PASSWORD='hospital123'
$env:HOSPITAL_JWT_SECRET='your-secret'
$env:HOSPITAL_WX_DEMO='true'
java -jar hospital-backend/target/hospital-backend-1.0.0-SNAPSHOT.jar --spring.profiles.active=prod

# 网关
node execution/scripts/local-gateway.mjs
# 浏览器: http://127.0.0.1:8088/admin/  http://127.0.0.1:8088/user/
```

生产环境将 `local-gateway.mjs` 替换为 `execution/nginx/hospital.conf` 即可。
