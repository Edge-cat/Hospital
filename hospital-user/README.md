# 东软云医院 HIS - 用户端

面向 **患者** 的 Web 在线服务平台，提供挂号、预约、缴费等自助就医功能。

## 端口

`5175` — `npm run dev`

访问 http://localhost:5175

## 功能页面

| 页面 | 功能 |
|------|------|
| 首页 | 快捷入口、科室推荐、公告 |
| 在线挂号 | 选科室/医生，提交挂号 |
| 在线预约 | 选日期时段预约 |
| 在线缴费 | 待缴费/已缴费、在线支付 |
| 科室介绍 | 科室列表与详情 |
| 医院公告 | 公告列表与详情 |
| 个人中心 | 账户信息、快捷菜单 |
| 我的挂号/预约 | 记录查询、退号/取消 |
| 就诊记录 | 历史诊疗查询 |

## 演示账号

`patient` / `123456`

## 三端说明

| 端 | 目录 | 使用者 |
|----|------|--------|
| 管理端 | `hospital-admin` | 管理员、医生、护士等工作人员 |
| 用户端 | `hospital-user`（本目录） | 患者（Web 浏览器） |
| 小程序端 | `hospital-mini` | 患者（微信小程序） |
